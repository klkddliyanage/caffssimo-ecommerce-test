# Public assets

Add your photos and logos here. They will be served from the site root.

**Examples:**
- `public/logo.svg` → use as `/logo.svg` in your code
- `public/images/hero.jpg` → use as `/images/hero.jpg`
- `public/photos/branch-downtown.jpg` → use as `/photos/branch-downtown.jpg`

In React/JSX, reference them with a leading slash, e.g. `src="/logo.svg"` or `src="/images/hero.jpg"`.
